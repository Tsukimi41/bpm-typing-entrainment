# 実行方法

このリポジトリは、Vite + React + TypeScript で動く静的Webアプリです。サーバー起動やデータベース設定は不要で、ローカルでは開発サーバーを立てて確認し、必要に応じて静的ビルドを行います。

## 前提条件

- Node.js 18 以上を推奨します。
- npm が使える環境を想定しています。
- タイピング実験は PC のハードウェアキーボードで実施してください。
- スマートフォンやタブレットではブロックされます。

## 初回セットアップ

1. リポジトリのルートに移動します。
2. 依存関係をインストールします。

```bash
npm install
```

## 開発サーバーの起動

画面を確認しながら開発する場合は、次のコマンドを実行します。

```bash
npm run dev
```

実行後、ターミナルに表示されるローカルURLをブラウザで開きます。通常は `http://localhost:5173` です。

## 本番用ビルド

配布や Vercel などの静的ホスティングに載せる前に、必ずビルドを通します。

```bash
npm run build
```

ビルド成功後、`dist/` に静的ファイルが生成されます。

## ビルド結果の確認

生成物をローカルで確認したい場合は、次を使います。

```bash
npm run preview
```

`preview` は `dist/` をもとに本番相当の表示を確認するためのコマンドです。

## うまく動かないとき

- `npm install` が失敗する場合は、Node.js のバージョンを確認してください。
- 画面がモバイル扱いでブロックされる場合は、ウィンドウ幅を広げるか PC ブラウザを使ってください。
- 入力が反応しない場合は、テキストボックスではなく画面全体にキー入力を送る仕様であることを確認してください。
- クリップボードコピーが失敗する場合は、HTTPS もしくはローカル環境であること、またブラウザの権限設定を確認してください。

## ディレクトリの見方

- `src/` にはアプリ本体があります。
- `src/data/trialTexts.ts` に課題文の生成ロジックがあります。
- `src/lib/typingUtils.ts` に CSV 出力や判定補助があります。
- `src/styles.css` に最小限のスタイルがあります。
